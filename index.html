HTML


Tables


Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Data Cell The data found within the table row.
<th> Header Cell The data found withing the <thead>.
<tfoot> Table Foot The set of rows defining the footer in a table.

Forms

Elements

Element Type Description
<form> Form Used to collect and send data to another source.
<input> *Variety* Can render a variety of input fields through type=.
<label> Label/
Identification		 Provides identification for a specific <input>. The <label>'s for="#" attribute should match the <input>'s id="#" attribute.
<textarea> Text-box Creates a text-box for multi-line input. Supports rows= and cols= attributes.
<select> Select Creates a dropdown list. You can create different options within the <option> </option> element within the <select> element.
<option> Options to Select Used within the <select> element to create various options.
<datalist> Datalist This element is created under the <input list="#"> element. The id= attribute in the datalist should match the value of the list= in the input. Used to store a list of options by creating a value= in the <option value="#"/> element.

type=

type= Name Description
"checkbox" Chackbox Allows the user to check multiple boxes under the same name= attribute.
"number" Number Type Allows the user to enter only numbers and a few special characters. Key-value pair will have name: value when the form is submitted. Contains name="#" attribute.
"range" Range Type Inserts a slider. Min and max value can be set with the min= and max= attributes. The steps inbetween the values can be set with the step= attribute.
"text" Text Type Renders a single row input field that users can type text inside. Can be used for username field.
"radio" Radio Button Type Renders a singe radio button. Multiple radio butons of the same topic is given the same name="#". Only one value="#" attribute can be selected by the user.
"submit" Submit Button Renders a submit button. Default states Submit, however, the name can be changed through the value="#" attribute.
"password" Password Type Renders a single row input field that censores text to protext sensitive information. The type and name attribute is sent out as a key-value pair.

Type Name Description

CSS


Display and Positioning


Miscellaneous

Property Values Description
z-index: #; Uses integer values that can be positive or negative. This decides which element is in the foreground and background.
float: left;
right; 		Determins how far left or right an element should float within its parent element. The width: of the container must be specified or the element will assume the full width of its containing element.

left; ← Content will float to the left of the container.

right; ← Content will float to the right of the container.
clear: right;
left;
both;
none; 		Specifies how an element should behave when it bumps into another element within the same containing element. The clear: property is used with the float: properly. This determines which side the floating elements are allowed to float.

right; ← No other elements within the same containing element are allowed to float on the left side of this element.

left; ← No other elements within the same containing element are allowed to float on the right side of this element.

both; ← No other elements within the same containing element are allowed to float on either side of this element.

none; ← Elements within the same containing element are allowed to float on both sides of this element.

Positioning

Property Values/Name Description
position:
  • fixed;
  • relative;
  • absolute;
  • static;
 Uses integer values that can be positive or negative. This decides which element is in the foreground and background.
fixed; Fixed Enables the user to scroll through the web page and still access the element. Mainly used with the <nav> element.
relative; Relative Enables an element to be positioned relative to where it would have been originally been on a web page. Offset proprerties can be used to adjust the element. If no values are applied, the element's position: property behaves like the static; value.
absolute; Absolute Removes an element from the document flow. Positions element to be positioned relative to its closest parent element that is positioned with relative; or absolute;. Positioning attributes can be applied: top:, right:, bottom:, and left:.
static; Static This is the default positioning of a given element.

Positioning Attributes

Name Description
top: How far from the top.
right: How far from the right.
bottom: How far from the bottom.
left: How far from the left.

display: Property

Value Description
block; Takes up the full width of their container with <br> before and after. Height and width can be adjusted.
inline; Takes up as little space as possible, flow horizontally. Height and width cannot be adjusted.
inline-block; Can appear next to each other. Can have width and height manually adjusted.
